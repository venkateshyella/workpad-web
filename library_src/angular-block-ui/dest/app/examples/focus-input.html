To prevent forms from re-submitting data, focused elements will be blured whenever a containing block is active. Unless the focus has been changed by the user while the (non-global) block was active, the focus is restored whenever the block has finished.

Restoring focus to any input element other than the submit button is not possible in any version of Internet Explorer. All other major browsers should be able to restore the focus to any previous element.
