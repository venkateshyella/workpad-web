angular-block-ui

Setting the main block-ui instance manually.

I am the main block instance

By default the block-ui module marks the body element as the main block instance. But in some cases this behaviour is not desired. When for instance the body element is not accessible or your ng-app directive is a child element of the body element.

It is however quite simple to relocate the main block-ui instance by simply marking a different element as the main instance.

<!-- Mark the desired containing element as the main block instance -->
<div block-ui="main" class="block-ui-main">
  <p>Lorem ipsum ... </p>
</div>
angular.module('myApp').config(function(blockUIConfig) {
  // Tell blockUI not to mark the body element as the main block scope.
  blockUIConfig.autoInjectBodyBlock = false;
});
